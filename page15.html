Exercice : Création de cartes de membre avec Excel et Word

Objectif : Apprendre à utiliser Excel pour stocker des données de membres et Word pour créer des cartes de membre en utilisant la fonctionnalité de publipostage.

Durée estimée : 15 minutes

Étape 1 : Création du fichier Excel

  • Ouvrez Excel.
  • Dans une nouvelle feuille, saisissez les données suivantes dans les colonnes A à C :
    • A1 : "Nom"
    • B1 : "Prénom"
    • C1 : "Âge"
  • Entrez les informations suivantes pour trois membres dans les lignes 2 à 4 :
    • A2 : "Dupont" | B2 : "Jean" | C2 : "28"
    • A3 : "Martin" | B3 : "Claire" | C3 : "35"
    • A4 : "Lefebvre" | B4 : "Marc" | C4 : "42"
  • Sauvegardez le fichier sous le nom "Membres.xlsx".

Étape 2 : Création de la carte de membre dans Word

  • Ouvrez Word.
  • Créez un nouveau document.
  • Allez dans l'onglet "Publipostage" et cliquez sur "Démarrer la fusion et le publipostage".
  • Sélectionnez "Documents d'adhésion".
  • Cliquez sur "Sélection des destinataires" puis "Utiliser une liste existante".
  • Ouvrez le fichier "Membres.xlsx" que vous avez créé précédemment.
  • Sélectionnez la feuille contenant les données des membres et cliquez sur "OK".
  • Dans le document Word, saisissez le modèle de la carte de membre comme suit :
    • Carte de Membre

Nom : « Nom »
Prénom : « Prénom »
Âge : « Âge »

Bienvenue dans notre association !
  • Pour insérer les champs de fusion :
    • Cliquez sur "Insérer un champ de fusion" dans l'onglet "Publipostage".
    • Sélectionnez "Nom", "Prénom", et "Âge" dans la liste des champs disponibles.
    • Les champs seront insérés à l'endroit où vous avez placé les guillemets.
  • Pour prévisualiser les cartes :
    • Cliquez sur "Aperçu des résultats" dans l'onglet "Publipostage".
    • Vous pouvez naviguer entre les enregistrements pour voir à quoi ressemblent les cartes pour chaque membre.
  • Une fois satisfait, cliquez sur "Terminer et fusionner" puis "Imprimer les documents" pour imprimer les cartes de membre.

Ce petit exercice vous permet de découvrir comment utiliser Excel pour stocker des données et Word pour créer des documents personnalisés à l'aide de la fonctionnalité de publipostage.
