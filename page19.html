Exercice : Création d'une Liste Déroulante dans Excel

Objectif :

Apprendre à créer une liste déroulante dans une cellule Excel pour faciliter la saisie des données.

Durée : 15 minutes

Étape 1 : Préparation des données

  1. Ouvrir Excel.
  2. Dans une nouvelle feuille, à partir de la cellule A1, entrer les options que vous voulez voir apparaître dans la liste déroulante. Par exemple :
    • A1 : handicap moteur
    • A2 : handicap auditif
    • A3 : handicap visuel
    • A4 : handicap mental
  3. Sélectionnez la plage de cellules A1:A4.

Étape 2 : Nommer la plage de cellules (optionnel)

  1. Optionnel : Pour plus de clarté, vous pouvez nommer cette plage. Cliquez dans la zone de nom en haut à gauche (où vous voyez généralement l'adresse de la cellule) et tapez un nom pour cette plage, comme "ListeHandicaps", puis appuyez sur Entrée.

Étape 3 : Créer la liste déroulante

  1. Sélectionnez la cellule dans laquelle vous souhaitez insérer la liste déroulante, par exemple, la cellule C1.
  2. Allez dans l'onglet Données du ruban.
  3. Cliquez sur Validation des données (ou Validation dans certaines versions).
  4. Dans la fenêtre qui s'ouvre, sous l'onglet Options, choisissez Liste dans la section Autoriser.
  5. Dans la zone Source, entrez soit directement les options séparées par des virgules (par exemple, handicap moteur,handicap auditif,handicap visuel,handicap mental), soit entrez le nom de la plage si vous avez suivi l'étape 2 (par exemple, =ListeHandicaps).
  6. Cliquez sur OK.

Étape 4 : Tester la liste déroulante

  1. Cliquez sur la cellule C1. Vous verrez une flèche apparaître à côté de la cellule.
  2. Cliquez sur la flèche pour voir et sélectionner les options disponibles dans la liste déroulante.

Étape 5 : Copier la liste déroulante (optionnel)

  1. Si vous souhaitez que la même liste déroulante apparaisse dans d'autres cellules, sélectionnez la cellule C1.
  2. Utilisez la poignée de recopie (coin inférieur droit de la cellule) pour copier la liste déroulante vers d'autres cellules.

Résultat attendu :

À la fin de cet exercice, vous aurez une cellule avec une liste déroulante contenant les options spécifiées, ce qui facilite la saisie des données sans erreurs de frappe.

Cet exercice est conçu pour durer environ 15 minutes, vous permettant de comprendre et d'appliquer la création de listes déroulantes dans Excel.
