TP : Création d'un Graphique avec Étiquettes de Données et Mise à Jour Dynamique

Objectif :

Apprendre à créer un graphique simple dans Excel, ajouter un titre, personnaliser les couleurs, insérer des étiquettes de données, et observer les changements dans le graphique après modification des données.

Étapes du TP :

  1. Créer un tableau de données :
    • Ouvrez une nouvelle feuille Excel.
    • Entrez les données suivantes :
      • Mois Ventes (en unités)
      Janvier =120
      Février =150
      Mars =130
      Avril =160
      Mai =180
    • Sélectionnez le tableau de données (A1:B6).
  2. Insérer un graphique :
    • Allez dans l'onglet Insertion.
    • Cliquez sur Graphiques dans le groupe Graphiques.
    • Choisissez un Graphique à barres.
  3. Modifier le graphique :
    • Cliquez sur le graphique pour le sélectionner.
    • Dans l'onglet Création, choisissez un style de graphique parmi ceux proposés.
    • Accédez à l'onglet Création :
    • Ajouter un titre au graphique :
    • Dans l'onglet création ou Conception, cherchez le groupe Éléments du graphique.
    • Cliquez sur Ajouter un élément de graphique.
    • Dans le menu déroulant qui apparaît, sélectionnez Titre du graphique.
    • Choisissez Au-dessus du graphique si vous souhaitez que le titre apparaisse au-dessus du graphique.
    • Modifier le titre :
    • Un champ de texte "Titre du graphique" apparaîtra sur le graphique.
    • Cliquez sur ce champ pour le sélectionner, puis tapez le titre souhaité, par exemple "Ventes Mensuelles".
  4. Personnalisation des éléments du graphique :
    • Cliquez sur l'axe des Y (valeurs des ventes) et formatez l'axe pour que les valeurs soient affichées en gras.
    • Modifiez les couleurs des barres en choisissant une couleur différente pour chaque mois.
  5. Ajouter des étiquettes de données :
    • Sélectionnez le graphique.
    • Cliquez sur Ajouter un élément de graphique > Étiquettes de données > Centre pour ajouter les valeurs directement sur les barres du graphique.
    • Vérifiez que les valeurs des ventes apparaissent au-dessus de chaque barre.
  6. Modifier les valeurs du tableau et observer les changements :
    • Changez les valeurs de ventes dans le tableau de données, par exemple :
      • Janvier : 500
      • Février : 470
      • Mars : 10
    • Observez comment le graphique se met automatiquement à jour pour refléter les nouvelles valeurs.
  7. Enregistrer votre travail :
    • Enregistrez le fichier sous le nom "Graphique_Ventes_Avec_Mise_A_Jour.xlsx".

Résultat attendu :

Un graphique en colonnes représentant les ventes mensuelles avec un titre personnalisé, des couleurs différentes pour chaque mois, des étiquettes de données affichant les valeurs des ventes, et une mise à jour dynamique du graphique lorsque les valeurs du tableau sont modifiées.
