Partie 1 : Utilisation des Filtres

Dans tab_initial(2) faire :

Étape 1 : Appliquer un Filtre
  1. Sélectionnez les données :
  2. Cliquez sur n'importe quelle cellule du tableau (par exemple, sur la cellule `A1`).
  3. Ajouter des filtres :
  4. Allez dans l'onglet "Données".
  5. Cliquez sur "Filtrer". Des petites flèches apparaissent dans les en-têtes de colonnes.
Étape 2 : Filtrer par Service
  1. Cliquez sur la flèche à côté de l'en-tête "service".
  2. Décochez tout, puis cochez uniquement la case "Une visite à domicile".
  3. Cliquez sur "OK" pour appliquer le filtre.
  4. Résultat : Vous ne verrez que les lignes où le service est "Une visite à domicile".
  5. Créer une nouvelle feuille nommée résultat : saisir « nombre des visites à domicile » = (le chiffre que vous avez trouvé)
Étape 4 : Filtrer par Travailleur Social (ts)
  1. Enlevez le filtre sur le service (Cliquez sur "Effacer le filtre").
  2. Cliquez sur la flèche à côté de l'en-tête "ts".
  3. Décochez tout, puis cochez uniquement la case "ts1".
  4. Cliquez sur "OK" pour appliquer le filtre.
  5. Résultat : Vous ne verrez que les activités réalisées par `ts1`.
  6. Dans la feuille nommée résultat : saisir « nombre des activités réalisées par ts1 » = (le chiffre que vous avez trouvé)
Étape 5 : Filtre Combiné (Service et Travailleur Social)
  1. Enlevez le filtre sur le travailleur social (Cliquez sur "Effacer le filtre").
  2. Cliquez sur la flèche à côté de l'en-tête "service".
  3. Décochez tout, puis cochez uniquement la case "Animation d'ateliers collectifs".
  4. Cliquez sur la flèche à côté de l'en-tête "ts".
  5. Décochez tout, puis cochez uniquement la case "ts4".
  6. Cliquez sur "OK" pour appliquer le filtre.
  7. Résultat : Vous ne verrez que les "Animations d'ateliers collectifs" réalisées par `ts4`.
  8. Dans la feuille nommée résultat : saisir « nombre des Animations d'ateliers collectifs réalisées par ts4 » = (le chiffre que vous avez trouvé)

Partie 2 : Introduction aux Tableaux Croisés Dynamiques

Étape 1 : Créer un Tableau Croisé Dynamique (TCD)
  1. Sélectionner les données :
  2. Cliquez sur une cellule quelconque de votre tableau.
  3. Allez dans l'onglet "Insertion" et cliquez sur "Tableau croisé dynamique".
  4. Une fenêtre apparaît. Assurez-vous que la plage de données est correcte et sélectionnez "Nouvelle feuille de calcul".
  5. Cliquez sur "OK".
