Contexte :

Étape 1 : Filtre Simple

  1. Objectif : Filtrer tous les produits de la catégorie "Électronique".
  2. Instructions :
    • Sélectionnez la colonne Catégorie.
    • Appliquez un filtre pour afficher uniquement les produits de la catégorie "Électronique".
    • Copiez les résultats filtrés dans une nouvelle feuille nommée "Électronique".

Étape 2 : Filtre avec Condition Numérique

  1. Objectif : Filtrer les produits de la catégorie "Électronique" avec un prix supérieur à 100€.
  2. Instructions :
    • Utilisez le filtre déjà appliqué sur la catégorie "Électronique".
    • Ajoutez un filtre sur la colonne Prix pour afficher uniquement les produits dont le prix est supérieur à 100€.
    • Copiez les résultats filtrés dans une nouvelle feuille nommée "Électronique >100€".

Étape 3 : Filtre avec Condition de Texte

  1. Objectif : Filtrer les produits de la catégorie "Électronique" dont le prix est supérieur à 100€ et le vendeur est "TechStore".
  2. Instructions :
    • Continuez avec le filtre précédent (Catégorie = "Électronique" et Prix > 100€).
    • Ajoutez un filtre sur la colonne Vendeur pour afficher uniquement les produits vendus par "TechStore".
    • Copiez les résultats filtrés dans une nouvelle feuille nommée "TechStore >100€".

Étape 4 : Filtre par Date

  1. Objectif : Filtrer les produits de la catégorie "Électronique" dont le prix est supérieur à 100€, vendus par "TechStore", et ajoutés en 2023.
  2. Instructions :
    • Continuez avec les filtres précédents.
    • Ajoutez un filtre sur la colonne Date d'Ajout pour afficher uniquement les produits ajoutés en 2023.
    • Copiez les résultats filtrés dans une nouvelle feuille nommée "2023 TechStore >100€".

Étape 5 : Filtre Avancé Composé

  1. Objectif : Filtrer les produits de la catégorie "Électronique" dont le prix est supérieur à 100€, avec une quantité en stock inférieure à 50, vendus par "TechStore", et ajoutés en 2023.
  2. Instructions :
    • Continuez avec les filtres précédents.
    • Ajoutez un filtre sur la colonne Quantité en Stock pour afficher uniquement les produits avec une quantité en stock inférieure à 50.
    • Copiez les résultats filtrés dans une nouvelle feuille nommée "Stock <50".

Résultat attendu :

À la fin de l'exercice, vous aurez cinq nouvelles feuilles dans votre classeur Excel :

  • "Électronique"
  • "Électronique >100€"
  • "TechStore >100€"
  • "2023 TechStore >100€"
  • "Stock <50"

Chaque feuille contiendra un ensemble de données filtrées correspondant aux critères de chaque étape. Cet exercice permet de comprendre l'utilisation des filtres simples et composés dans Excel.
