Exercice : Tableau avec calcul d'âge en utilisant la date actuelle

Étape 1 : Création du tableau

  1. Ouvrir Excel.
  2. Dans la cellule A1, taper "Nom".
  3. Dans la cellule B1, taper "Prénom".
  4. Dans la cellule C1, taper "Date de Naissance".
  5. Dans la cellule D1, taper "Date Actuelle".
  6. Dans la cellule E1, taper "Âge (en années)".

Étape 2 : Saisie des données

  1. Dans les cellules A2 à A7, entrer les noms de 6 personnes.
  2. Dans les cellules B2 à B7, entrer les prénoms correspondants.
  3. Dans les cellules C2 à C7, entrer les dates de naissance (au format jj/mm/aaaa).

Étape 3 : Insérer la date actuelle

  1. Dans la cellule D2, saisir la formule suivante :
    2. =AUJOURD'HUI()
  2. Appuyer sur Entrée. La cellule D2 doit maintenant afficher la date d'aujourd'hui.
  3. Copier la formule de la cellule D2 vers D3 jusqu'à D7 pour remplir la date actuelle pour les autres lignes.

Étape 4 : Calculer l'âge

  1. Dans la cellule E2, saisir la formule suivante pour calculer l'âge :
    2. =(D2-C2)/365,25

    Cette formule soustrait la date de naissance de la date actuelle, puis divise le résultat par 365,25 (pour tenir compte des années bissextiles).

  2. Appuyer sur Entrée. La cellule E2 doit maintenant afficher l'âge en années, y compris les fractions.
  3. Copier la formule de la cellule E2 vers E3 jusqu'à E7 pour calculer l'âge des autres personnes.
  4. Optionnel : Si vous voulez arrondir l'âge à l'année la plus proche, vous pouvez utiliser la formule suivante dans la colonne E :
    5. =ARRONDI((D2-C2)/365,25; 0)

    Cette formule arrondira l'âge au nombre entier le plus proche.

Étape 5 : Trier les données

  1. Sélectionner les données de A1 à E7 (inclus).
  2. Aller dans l'onglet "Données".
  3. Cliquer sur "Trier".
  4. Choisir de trier par "Date de Naissance" ou "Âge (en années)" selon votre choix.
  5. Appuyer sur "OK" pour appliquer le tri.

Étape 6 : Calcul de l'âge minimum et maximum

  1. Dans la cellule G1, taper "Âge Minimum".
  2. Dans la cellule H1, taper "Âge Maximum".
  3. Dans la cellule G2, entrer la formule suivante pour trouver l'âge minimum :
    4. =MIN(E2:E7)
  4. Dans la cellule H2, entrer la formule suivante pour trouver l'âge maximum :
    5. =MAX(E2:E7)
  5. Appuyer sur Entrée après chaque formule. Les cellules G2 et H2 doivent afficher respectivement l'âge minimum et maximum parmi les personnes listées.

Étape 7 : Sauvegarde du fichier

  1. Aller dans "Fichier" puis "Enregistrer sous".
  2. Choisir un emplacement pour enregistrer le fichier.
  3. Donner un nom au fichier (par exemple, "Exercice Calcul d'Âge avec Date Actuelle").
  4. Cliquer sur "Enregistrer".
